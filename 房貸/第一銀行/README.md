# 第一銀行 - 房屋轉貸/增貸資訊

## 產品/方案名稱
- 購屋貸款（含轉貸）
- 房屋修繕貸款（含增貸）
- 理財型房貸

## 適用對象/申請條件
- 年滿20歲至65歲本國國民
- 具有正當職業及穩定收入
- 信用紀錄良好
- 房屋產權清楚，可供設定抵押權

## 利率區間/計價方式
- **機動利率**：本行定儲利率指數加碼
- 利率區間：約 2.06% ~ 3.8%（依個人條件核定）
- 提供**一段式**、**分段式**利率選擇
- **固定利率**方案：部分專案提供

## 費用說明
- **開辦費**：約 NT$5,000 ~ NT$12,000（或貸款金額一定比例）
- **鑑價費**：約 NT$3,000 ~ NT$6,000
- **代書費**：約 NT$8,000 ~ NT$12,000
- **提前清償違約金**：綁約期內提前清償需支付違約金
- **帳戶管理費**：依專案而定
- **火險及地震險**：依房屋價值投保

## 可貸成數、年限、寬限期
- **可貸成數**：最高80%（視房屋鑑價及個人條件）
- **貸款年限**：最長30年
- **寬限期**：最長3年（依核定條件）

## 申請流程/所需文件

### 申請流程
1. 提出申請並備妥相關文件
2. 銀行進行徵信調查及房屋鑑價
3. 審核評估
4. 核准並通知結果
5. 對保及簽訂契約
6. 抵押權設定登記
7. 撥款

### 所需文件
- 身分證、戶口名簿或戶籍謄本
- 收入證明（薪轉存摺、扣繳憑單、財務報表等）
- 土地及建物所有權狀
- 原貸款繳款紀錄證明
- 印鑑證明及印鑑章
- 其他銀行要求之文件
