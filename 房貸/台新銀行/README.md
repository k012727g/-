# 台新銀行 - 房屋轉貸/增貸資訊

## 產品/方案名稱
- 房屋貸款（含轉貸）
- 房屋增貸
- 理財型房貸
- 二胎房貸

## 適用對象/申請條件
- 年滿20歲至65歲之本國國民
- 具有正當職業及穩定收入
- 信用狀況良好，無重大債信瑕疵
- 房屋產權清楚，可供設定抵押

## 利率區間/計價方式
- **機動利率**：依本行指標利率加碼計算
- 利率區間：約 2.06% ~ 4.0%（依個人條件核定）
- 提供**一段式**、**分段式**、**混合式**利率方案
- 利率隨市場指標變動

## 費用說明
- **開辦費**：約 NT$5,000 ~ NT$15,000（或貸款金額一定比例）
- **鑑價費**：約 NT$3,000 ~ NT$6,000
- **代書費**：約 NT$8,000 ~ NT$15,000
- **提前清償違約金**：綁約期間內提前還款需支付違約金
- **帳戶管理費**：部分專案可能收取
- **火險、地震險**：依保險公司規定

## 可貸成數、年限、寬限期
- **可貸成數**：最高85%（依房屋鑑價及信用條件）
- **貸款年限**：最長40年（部分專案）
- **寬限期**：最長5年（依專案及核定條件）

## 申請流程/所需文件

### 申請流程
1. 線上或臨櫃申請
2. 提供所需文件
3. 銀行進行徵信及鑑價
4. 審核評估
5. 核准通知
6. 對保簽約
7. 抵押權設定
8. 撥款

### 所需文件
- 身分證、健保卡或駕照
- 收入證明（薪轉存摺、扣繳憑單等）
- 土地及建物所有權狀影本
- 原房貸繳款證明
- 印鑑證明及印鑑章
- 戶籍謄本
- 其他銀行要求文件
