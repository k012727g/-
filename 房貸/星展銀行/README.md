# 星展銀行 - 房屋轉貸/增貸資訊

## 產品/方案名稱
- 房屋貸款（含轉貸）
- 房屋增貸
- 理財型房貸

## 適用對象/申請條件
- 年滿20歲至65歲之本國國民或外國人
- 有正當職業及穩定收入來源
- 信用狀況良好
- 房屋產權清楚，可設定第一順位抵押權

## 利率區間/計價方式
- **機動利率**：依本行指標利率加碼計算
- 利率區間：約 2.20% ~ 4.0%（依個人條件）
- **一段式利率**或**分段式利率**
- 利率隨指標調整

## 費用說明
- **開辦費**：約 NT$8,000 ~ NT$15,000（或貸款金額 × 0.5% ~ 1%）
- **鑑價費**：約 NT$3,000 ~ NT$6,000
- **代書費**：約 NT$10,000 ~ NT$15,000
- **提前清償違約金**：綁約期內提前清償需支付違約金
- **帳管費**：部分專案收取
- **火險及地震險**：依房屋價值投保

## 可貸成數、年限、寬限期
- **可貸成數**：最高80%（依鑑價及個人條件）
- **貸款年限**：最長30年
- **寬限期**：最長3年（依核定條件）

## 申請流程/所需文件

### 申請流程
1. 諮詢及初步評估
2. 提出申請並檢附文件
3. 銀行徵信及房屋鑑價
4. 審核及核准
5. 對保及簽約
6. 地政設定
7. 撥款

### 所需文件
- 身分證、第二身分證明文件
- 收入證明（薪資轉帳、扣繳憑單、財務報表等）
- 土地及建物所有權狀影本
- 原房貸繳款證明
- 印鑑證明及印鑑章
- 戶籍謄本
- 其他銀行指定文件
