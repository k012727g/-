# 台灣各家銀行房屋轉增貸資訊

本資料夾整理台灣各家銀行房屋轉增貸（房屋轉貸、增貸）的公開資訊，提供統一格式的資料模板，方便比較各家銀行的房貸產品。

## 資料內容

每家銀行的資料包含以下項目：

1. **產品/方案名稱** - 銀行提供的房貸產品或方案名稱
2. **適用對象與申請條件** - 哪些人可以申請、需要符合什麼條件
3. **利率區間、計價方式** - 利率範圍、固定或浮動利率、計息方式
4. **費用細項** - 手續費、鑑價費、代書費等相關費用
5. **提前清償規則** - 提前還款的限制或違約金規定
6. **可貸成數、年限、寬限期** - 最高貸款成數、貸款年限、是否有寬限期
7. **申請流程** - 申請步驟說明
8. **所需文件** - 申請時需要準備的文件清單
9. **官方來源連結** - 銀行官方網站相關資訊連結

## 使用說明

1. 各銀行資料夾內有 README.md 檔案，包含該銀行的房貸資訊
2. 標示「*待補充*」的項目表示資料尚未收集完成
3. 所有資訊以各銀行官方公開資料為準
4. 實際申請條件與利率可能因個人信用狀況、擔保品價值等因素而異

## 注意事項

- 本資料僅供參考，實際貸款條件請以各銀行最新公告為準
- 建議申請前直接向銀行確認最新資訊
- 房貸利率與條件會隨市場變動而調整
