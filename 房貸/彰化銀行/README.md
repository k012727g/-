# 彰化銀行 - 房屋轉貸/增貸資訊

## 產品/方案名稱
- 購屋貸款（含轉貸）
- 房屋修繕貸款（含增貸）
- 理財型房貸

## 適用對象/申請條件
- 年滿20歲之本國國民
- 有固定職業及穩定收入
- 信用紀錄良好
- 房屋需可設定第一順位抵押權

## 利率區間/計價方式
- **機動利率**：依本行定儲指數利率加碼
- 利率區間：約 2.06% ~ 4.0%（依個人條件核定）
- 提供**一段式**及**分段式**計價方式
- 利率隨指標利率調整

## 費用說明
- **開辦費**：約 NT$5,000 ~ NT$10,000（或貸款金額一定比例）
- **鑑價費**：約 NT$3,000 ~ NT$5,000
- **代書費**：約 NT$8,000 ~ NT$12,000
- **提前清償違約金**：綁約期內提前清償需支付
- **火險及地震險保費**：依房屋價值計算

## 可貸成數、年限、寬限期
- **可貸成數**：最高80%（依房屋鑑價及個人條件）
- **貸款年限**：最長30年
- **寬限期**：最長3年（依核定條件）

## 申請流程/所需文件

### 申請流程
1. 諮詢及初步評估
2. 填寫申請書並備妥文件
3. 銀行徵信及鑑價
4. 審核及核准
5. 對保簽約
6. 設定抵押權
7. 撥款

### 所需文件
- 身分證、戶籍謄本
- 收入證明（薪轉證明、扣繳憑單等）
- 土地及建物所有權狀
- 原貸款繳息證明
- 印鑑證明及印鑑章
- 其他銀行要求文件
