# 渣打銀行 - 房屋轉貸/增貸資訊

## 產品/方案名稱
- 房屋貸款（含轉貸）
- 房屋增貸
- 理財型房貸

## 適用對象/申請條件
- 年滿20歲至65歲之本國國民或外國人
- 具有正當職業及穩定收入
- 信用紀錄良好
- 房屋產權清楚，可設定抵押權

## 利率區間/計價方式
- **機動利率**：依本行指標利率加碼
- 利率區間：約 2.20% ~ 4.0%（依個人條件核定）
- **一段式利率**或**分段式利率**可選
- 利率隨市場指標調整

## 費用說明
- **開辦費**：約 NT$8,000 ~ NT$15,000（或貸款金額 × 0.5% ~ 1%）
- **鑑價費**：約 NT$3,000 ~ NT$6,000
- **代書費**：約 NT$10,000 ~ NT$15,000
- **提前清償違約金**：綁約期內提前清償需支付違約金
- **帳管費**：部分專案收取
- **火險及地震險**：依房屋價值投保

## 可貸成數、年限、寬限期
- **可貸成數**：最高80%（視鑑價及個人條件）
- **貸款年限**：最長30年
- **寬限期**：最長3年（依核定條件）

## 申請流程/所需文件

### 申請流程
1. 諮詢及評估
2. 提出申請並提供文件
3. 銀行審查及鑑價
4. 核准通知
5. 對保及簽約
6. 抵押權設定
7. 撥款

### 所需文件
- 身分證、第二身分證明文件
- 收入證明（薪資單、扣繳憑單等）
- 土地及建物所有權狀
- 原貸款繳款證明
- 印鑑證明及印鑑章
- 戶籍謄本
- 其他銀行要求文件
