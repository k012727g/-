# 華南銀行 - 房屋轉貸/增貸資訊

## 產品/方案名稱
- 購置住宅貸款（含轉貸）
- 修繕住宅貸款（含增貸）
- 理財型房貸

## 適用對象/申請條件
- 年滿20歲至65歲具完全行為能力之國民
- 具有正當職業及穩定收入來源
- 無不良信用紀錄
- 房屋所有權清楚，可設定第一順位抵押權

## 利率區間/計價方式
- **機動利率**：依本行指標利率加碼計算
- 利率區間：約 2.06% ~ 3.9%（視個人條件而定）
- **一段式利率**：全期固定加碼
- **分段式利率**：不同期間不同加碼
- 利率隨指標調整而變動

## 費用說明
- **開辦費**：約 NT$5,000 ~ NT$10,000（或貸款金額 × 0.5% ~ 1%）
- **鑑價費**：約 NT$3,000 ~ NT$6,000
- **代書費**：約 NT$10,000 ~ NT$15,000
- **提前清償違約金**：綁約期間提前償還需支付違約金
- **火險及地震險**：依房屋鑑價投保

## 可貸成數、年限、寬限期
- **可貸成數**：最高80%（依鑑價及信用條件核定）
- **貸款年限**：最長30年
- **寬限期**：最長3年（經審核核定）

## 申請流程/所需文件

### 申請流程
1. 諮詢及評估
2. 提出申請並提供所需文件
3. 銀行審查及房屋鑑價
4. 核准
5. 對保及簽約
6. 抵押權設定
7. 核貸撥款

### 所需文件
- 身分證正反面影本、戶籍謄本
- 收入證明文件（薪資單、扣繳憑單等）
- 土地及建物所有權狀影本
- 原房貸繳款證明
- 印鑑證明及印鑑章
- 其他銀行指定文件
