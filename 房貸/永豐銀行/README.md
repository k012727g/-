# 永豐銀行 - 房屋轉貸/增貸資訊

## 產品/方案名稱
- 房屋轉貸優惠專案
- 房屋增貸
- 理財型房貸
- 數位房貸

## 適用對象/申請條件
- 年滿20歲至65歲之本國國民
- 具有正當職業及穩定收入
- 信用紀錄良好
- 房屋產權清楚，可供設定抵押權

## 利率區間/計價方式
- **機動利率**：依本行定儲利率指數加碼
- 利率區間：約 2.06% ~ 3.9%（依個人條件）
- **一段式利率**：全期統一加碼
- **分段式利率**：分期調整加碼
- 提供線上快速試算
- 利率隨指標調整

## 費用說明
- **開辦費**：約 NT$5,000 ~ NT$10,000（或貸款金額 × 0.5% ~ 1%）
- **鑑價費**：約 NT$3,000 ~ NT$6,000
- **代書費**：約 NT$10,000 ~ NT$15,000
- **提前清償違約金**：綁約期內提前還款需支付違約金
- **帳戶管理費**：部分專案收取
- **火險、地震險**：依房屋價值投保

## 可貸成數、年限、寬限期
- **可貸成數**：最高85%（依房屋鑑價及個人條件）
- **貸款年限**：最長40年
- **寬限期**：最長5年（經核定）

## 申請流程/所需文件

### 申請流程
1. 線上/電話/臨櫃申請
2. 提供所需文件
3. 銀行徵信及房屋鑑價
4. 審核評估
5. 核准通知
6. 對保簽約
7. 地政設定
8. 撥款

### 所需文件
- 身分證正反面影本、第二證明文件
- 收入證明（薪資單、扣繳憑單、財務報表等）
- 土地及建物所有權狀影本
- 原貸款繳款證明
- 印鑑證明及印鑑章
- 戶籍謄本
- 其他銀行指定文件
