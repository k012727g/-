# 土地銀行 - 房屋轉貸/增貸資訊

## 產品/方案名稱
- 房屋轉貸貸款
- 購置及修繕住宅貸款（含增貸）

## 適用對象/申請條件
- 年滿20歲之中華民國國民
- 有正當職業及穩定收入
- 信用良好，無重大債信不良紀錄
- 原貸款已正常繳款一定期間

## 利率區間/計價方式
- **機動利率**：依本行定儲利率指數加碼
- 利率區間：約 2.06% ~ 3.8%（依個人條件而定）
- 計價方式：指標利率 + 加碼利率
- **一段式**或**分段式**利率可選

## 費用說明
- **開辦費**：約 NT$5,000 ~ NT$10,000（或貸款金額 × 0.5% ~ 1%）
- **鑑價費**：約 NT$3,000 ~ NT$6,000
- **代書費**：約 NT$10,000 ~ NT$15,000
- **提前清償違約金**：依合約條款，通常未滿1年需支付違約金
- **帳管費**：部分專案可能收取

## 可貸成數、年限、寬限期
- **可貸成數**：最高80%（依鑑價及信用條件）
- **貸款年限**：最長30年
- **寬限期**：最長3年（經審核後核定）

## 申請流程/所需文件

### 申請流程
1. 填寫申請書並檢附相關文件
2. 銀行進行徵信及房屋鑑價
3. 審核及核准
4. 對保、簽約
5. 設定抵押權登記
6. 撥款

### 所需文件
- 國民身分證、戶口名簿影本
- 收入證明文件（薪轉存摺、扣繳憑單等）
- 土地及建物所有權狀影本
- 原房貸繳款證明
- 印鑑證明
- 其他銀行要求文件
