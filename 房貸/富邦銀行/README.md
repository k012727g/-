# 富邦銀行 - 房屋轉貸/增貸資訊

## 產品/方案名稱
- 房屋轉貸優惠專案
- 房屋增貸
- 理財型房貸
- 數位房貸

## 適用對象/申請條件
- 年滿20歲至65歲之本國國民
- 有正當職業及穩定收入來源
- 信用狀況良好
- 房屋產權清楚，可設定第一順位抵押權

## 利率區間/計價方式
- **機動利率**：依本行指標利率加碼計算
- 利率區間：約 2.06% ~ 3.9%（依個人條件核定）
- **一段式利率**：全期加碼固定
- **分段式利率**：階段性加碼
- **混合式利率**：固定+機動組合
- 提供線上快速試算
- 利率隨指標調整

## 費用說明
- **開辦費**：約 NT$5,000 ~ NT$12,000（或貸款金額 × 0.5% ~ 1%）
- **鑑價費**：約 NT$3,000 ~ NT$6,000
- **代書費**：約 NT$10,000 ~ NT$15,000
- **提前清償違約金**：綁約期內提前清償需支付違約金
- **帳管費**：部分專案可能收取
- **火險及地震險**：依房屋價值投保

## 可貸成數、年限、寬限期
- **可貸成數**：最高85%（依鑑價及信用條件）
- **貸款年限**：最長40年
- **寬限期**：最長5年（依核定條件）

## 申請流程/所需文件

### 申請流程
1. 線上/電話/臨櫃諮詢
2. 填寫申請書並備妥文件
3. 銀行徵信及房屋鑑價
4. 審核及核准
5. 對保及簽約
6. 抵押權設定登記
7. 撥款

### 所需文件
- 身分證正反面影本
- 收入證明（薪轉存摺、扣繳憑單、營業執照等）
- 土地及建物所有權狀
- 原房貸繳款證明
- 印鑑證明及印鑑章
- 戶籍謄本
- 其他銀行要求文件
