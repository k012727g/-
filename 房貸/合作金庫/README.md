# 合作金庫 - 房屋轉貸/增貸資訊

## 產品/方案名稱
- 購屋貸款（含轉貸）
- 房屋修繕貸款（含增貸）

## 適用對象/申請條件
- 年滿20歲具有完全行為能力之本國國民
- 有正當職業及穩定收入來源
- 信用狀況良好
- 房屋所有權須完整，無產權糾紛

## 利率區間/計價方式
- **機動利率**：依本行定儲利率指數加碼計價
- 利率區間：約 2.06% ~ 4.0%（依個人條件）
- 提供**一段式**及**分段式**利率方案
- 利率隨市場指標調整

## 費用說明
- **開辦費**：NT$5,000 ~ NT$10,000 或貸款金額 × 0.5% ~ 1%
- **鑑價費**：約 NT$3,000 ~ NT$5,000
- **代書費**：約 NT$8,000 ~ NT$15,000
- **提前清償違約金**：綁約期間內提前還款需支付違約金
- **火險、地震險費用**：依保險公司規定

## 可貸成數、年限、寬限期
- **可貸成數**：最高80%（視鑑價及個人條件）
- **貸款年限**：最長30年（含寬限期）
- **寬限期**：最長3年

## 申請流程/所需文件

### 申請流程
1. 諮詢及評估
2. 填寫申請書並備妥文件
3. 銀行審核及鑑價
4. 核准通知
5. 對保簽約
6. 地政設定
7. 撥款完成

### 所需文件
- 身分證明文件
- 收入證明（薪資單、扣繳憑單、營利事業登記證等）
- 土地建物所有權狀
- 原貸款繳款證明
- 印鑑證明及印鑑章
- 戶籍謄本
- 其他銀行指定文件
