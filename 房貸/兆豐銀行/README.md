# 兆豐銀行 - 房屋轉貸/增貸資訊

## 產品/方案名稱
- 購屋貸款（含轉貸）
- 房屋修繕貸款（含增貸）
- 理財型房貸
- 二胎房貸

## 適用對象/申請條件
- 年滿20歲至65歲之本國國民或外國人
- 有正當職業及穩定收入
- 信用狀況良好
- 房屋產權清楚，可設定抵押權

## 利率區間/計價方式
- **機動利率**：依本行定儲利率指數加碼
- 利率區間：約 2.06% ~ 3.8%（依個人條件）
- **一段式**或**分段式**利率可選
- 部分專案提供**固定利率**期間

## 費用說明
- **開辦費**：約 NT$5,000 ~ NT$12,000（或貸款金額 × 0.5% ~ 1%）
- **鑑價費**：約 NT$3,000 ~ NT$6,000
- **代書費**：約 NT$10,000 ~ NT$15,000
- **提前清償違約金**：綁約期內提前清償需支付違約金
- **帳管費**：部分專案收取
- **火險及地震險**：依房屋價值投保

## 可貸成數、年限、寬限期
- **可貸成數**：最高80%（視鑑價及個人條件）
- **貸款年限**：最長30年
- **寬限期**：最長3年（經核定）

## 申請流程/所需文件

### 申請流程
1. 諮詢及評估
2. 提出申請並檢附文件
3. 徵信調查及房屋鑑價
4. 審核及核准
5. 對保及簽約
6. 地政設定
7. 撥款

### 所需文件
- 身分證、戶口名簿或戶籍謄本
- 收入證明（薪資單、扣繳憑單、財報等）
- 土地及建物所有權狀
- 原貸款繳款證明
- 印鑑證明及印鑑章
- 其他銀行指定文件
