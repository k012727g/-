# 台灣銀行 - 房屋轉貸/增貸資訊

## 產品/方案名稱
- 房屋轉貸貸款
- 房屋增貸

## 適用對象/申請條件
- 年滿20歲至65歲之中華民國國民
- 有正當職業及固定收入來源
- 信用狀況良好
- 原房貸已繳款一定期間（通常需滿1年以上）

## 利率區間/計價方式
- **機動利率**：依本行定儲利率指數加碼計算
- 利率區間：約 2.06% ~ 3.5%（依個人條件而定）
- 計價方式：指標利率 + 加碼利率
- 利率會隨指標利率調整而變動

## 費用說明
- **開辦費**：貸款金額 × 0.5% ~ 1%（最低約 NT$3,000 起）
- **鑑價費**：約 NT$3,000 ~ NT$6,000（依物件而定）
- **代書費**：約 NT$8,000 ~ NT$15,000
- **提前清償違約金**：
  - 未滿1年：違約金約貸款餘額 × 1%~2%
  - 1年以上：視合約條款而定
- **火險、地震險**：依房屋價值及保險公司報價

## 可貸成數、年限、寬限期
- **可貸成數**：最高80%（依房屋鑑價及申請人條件）
- **貸款年限**：最長30年
- **寬限期**：最長3年（依申請人條件核定）

## 申請流程/所需文件

### 申請流程
1. 諮詢評估
2. 提出申請並檢附文件
3. 房屋鑑價
4. 銀行審核
5. 對保簽約
6. 設定抵押權
7. 撥款

### 所需文件
- 身分證、戶口名簿或戶籍謄本
- 收入證明（薪資轉帳證明、扣繳憑單、營利事業登記證等）
- 原房貸繳款證明
- 土地及建物所有權狀影本
- 印鑑證明及印鑑章
- 其他銀行要求文件
