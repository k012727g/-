# 中國信託銀行 - 房屋轉貸/增貸資訊

## 產品/方案名稱
- 房屋轉貸優惠專案
- 房屋增貸
- 理財型房貸
- 二胎房貸

## 適用對象/申請條件
- 年滿20歲至65歲之本國國民
- 具有正當職業及穩定收入
- 信用狀況良好
- 房屋所有權清楚，可設定抵押權

## 利率區間/計價方式
- **機動利率**：依本行指標利率加碼計算
- 利率區間：約 2.06% ~ 4.0%（依個人條件）
- **一段式利率**：全期統一加碼
- **分段式利率**：階段性調整加碼
- **固定利率**：部分專案提供固定期間
- 利率隨指標調整

## 費用說明
- **開辦費**：約 NT$5,000 ~ NT$15,000（或貸款金額 × 0.5% ~ 1%）
- **鑑價費**：約 NT$3,000 ~ NT$6,000
- **代書費**：約 NT$10,000 ~ NT$15,000
- **提前清償違約金**：綁約期內提前清償需支付
- **帳管費**：部分專案收取
- **火險、地震險**：依房屋價值投保

## 可貸成數、年限、寬限期
- **可貸成數**：最高85%（視鑑價及個人條件）
- **貸款年限**：最長40年
- **寬限期**：最長5年（依專案核定）

## 申請流程/所需文件

### 申請流程
1. 線上/電話/分行諮詢
2. 提出申請並檢附文件
3. 銀行審查及房屋鑑價
4. 核准通知
5. 對保及簽訂契約
6. 抵押權設定登記
7. 核貸撥款

### 所需文件
- 身分證正反面影本、第二身分證明文件
- 收入證明（薪轉存摺、扣繳憑單、財務報表等）
- 土地及建物所有權狀影本
- 原貸款繳款證明
- 印鑑證明及印鑑章
- 戶籍謄本
- 其他銀行要求文件
