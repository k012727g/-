# 法律文件管理專案

這個專案用於管理和組織與法律案件相關的文件。

## 專案結構

```
.
├── Lawsuit_Invalid_Divorce/    # 離婚無效訴訟相關文件
│   ├── Maintenance_Civil_Statement_of_Defence.txt
│   └── 扶養費對話/
├── letters/                    # 存證信函及法律文書
│   ├── 存證信函範本.md
│   ├── 存證信函_房東取消門禁卡_20260125.md
│   └── 刑事告訴狀_強制罪_房東取消門禁_20260125.md
└── README.md
```

## 文件類別

### 訴訟文件 (Lawsuit Documents)
包含與訴訟案件相關的法律文書，如答辯狀、起訴狀等。

### 存證信函 (Certified Letters)
包含存證信函範本及實際發送的存證信函。

## 使用說明

1. 各類文件按照案件類型分類存放
2. 檔案名稱包含日期標記，格式為 `YYYYMMDD`
3. 文件使用 Markdown 格式以便於版本控制和閱讀

## 注意事項

- 本專案中的文件為個人法律文件，請勿轉載或分享
- 所有範本僅供參考，實際使用前請諮詢專業律師
