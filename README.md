# 個人法律文件管理專案

此專案用於管理與組織個人法律相關文件，包含訴訟資料與法律信函範本。

## 專案結構

```
.
├── README.md                    # 專案說明文件
├── Lawsuit_Invalid_Divorce/     # 訴訟相關文件
│   ├── Maintenance_Civil_Statement_of_Defence.txt  # 民事答辯狀
│   └── 扶養費對話/                                   # 扶養費相關對話記錄
└── letters/                     # 法律信函範本
    ├── 刑事告訴狀_強制罪_房東取消門禁_20260125.md  # 刑事告訴狀範本
    ├── 存證信函_房東取消門禁卡_20260125.md         # 存證信函範本
    └── 存證信函範本.md                              # 一般存證信函範本
```

## 使用說明

1. **訴訟文件**：存放於 `Lawsuit_Invalid_Divorce/` 資料夾
2. **信函範本**：存放於 `letters/` 資料夾

## 注意事項

- 請妥善保管個人資訊，避免洩露敏感資料
- 使用範本時，請根據實際情況填入正確資訊
- 建議定期備份重要文件
