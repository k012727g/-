# 法律文件管理

此專案用於管理與組織個人法律文件，包含租賃糾紛及家事訴訟相關文件。

## 專案結構

```
├── letters/                           # 存證信函與刑事告訴狀
│   ├── 存證信函範本.md                  # 通用存證信函範本
│   ├── 存證信函_房東取消門禁卡_20260125.md  # 租賃糾紛存證信函
│   └── 刑事告訴狀_強制罪_房東取消門禁_20260125.md  # 強制罪刑事告訴狀
│
└── Lawsuit_Invalid_Divorce/           # 離婚/扶養費訴訟相關文件
    ├── Maintenance_Civil_Statement_of_Defence.txt  # 扶養費民事答辯狀
    └── 扶養費對話/                      # 相關對話截圖
```

## 文件說明

### letters（存證信函）
- **存證信函範本.md** - 通用存證信函格式，可作為撰寫其他存證信函之參考
- **存證信函_房東取消門禁卡_20260125.md** - 針對房東取消社區門禁磁扣之存證信函
- **刑事告訴狀_強制罪_房東取消門禁_20260125.md** - 針對房東取消門禁構成強制罪之刑事告訴狀

### Lawsuit_Invalid_Divorce（離婚/扶養費訴訟）
- 扶養費民事答辯狀及相關證據資料

## 注意事項

本專案內容僅供個人使用與參考，非法律意見。涉及重大權益或複雜案件，建議洽詢專業律師
