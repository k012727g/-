# 生活文件管理專案 (Life Documents Management Project)

## 專案簡介 (Project Overview)

這是一個用於管理個人重要生活文件的專案，包括法律文件、信函、以及其他重要文件的整理與歸檔。

This is a project for managing important personal life documents, including legal documents, letters, and other important files for organization and archiving.

## 專案結構 (Project Structure)

```
.
├── Lawsuit_Invalid_Divorce/    # 訴訟相關文件 (Lawsuit-related documents)
│   └── 扶養費對話/              # 扶養費相關對話記錄 (Maintenance fee discussions)
├── letters/                     # 信函與存證信函 (Letters and certified letters)
├── README.md                    # 專案說明文件 (Project documentation)
└── .gitignore                   # Git 忽略文件設定 (Git ignore configuration)
```

## 文件分類 (Document Categories)

### 1. 訴訟文件 (Lawsuit Documents)
- 民事答辯狀 (Civil Statement of Defence)
- 相關證據與對話記錄 (Related evidence and conversation records)

### 2. 信函 (Letters)
- 刑事告訴狀 (Criminal complaints)
- 存證信函 (Certified letters)
- 信函範本 (Letter templates)

## 使用說明 (Usage Instructions)

1. **新增文件**: 根據文件類型放置於對應的資料夾中
2. **命名規範**: 建議使用格式 `文件類型_主題_日期.md` (如: 存證信函_房東取消門禁卡_20260125.md)
3. **隱私保護**: 請勿上傳包含個人敏感資訊的原始文件，建議去識別化處理

## 注意事項 (Notes)

- 本專案僅供個人文件管理使用
- 請定期備份重要文件
- 敏感資訊應妥善保管，避免外洩
